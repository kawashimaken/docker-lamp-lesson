# 使い方

Dockerが必要です。


## STEP 1
```bash
git clone [this depository]
cd [this depository]/
docker-compose up
```


## STEP2

ブラウザでhttp://localhost にアクセスしてください


# データベース

localhost:8080 から phpmyadmin が使えます。

phpmyadminからデータベースやテーブルの詳細を確認できます。
